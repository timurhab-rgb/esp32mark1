Хаааааааай, данный проект является прототипом автоматезированной системы проверки железнодорожных путей,
на данном этапе проектирования уже создан телеграм бот отвечающий за управление системой, а так же прописывается логика модели.

Сама система представляет из себя набор датчиков, данные с которых отправляюстся на микроконтроллер esp32, от куда в свою очередь эти данные 
передаются на прямую в телеграм бота(ссылка на бота: https://t.me/my_esp32_controller_bot)

По окончании проекта система должна будет уметь получать физические данные с датчиков и отправлять их по запросу пользователя в чат бота.
